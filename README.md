## 📝 Cómo Usar

### 1. Ingresar al sistema
- Ve al link arriba
- Ingresa tu email y contraseña
- Verás tu sitio en la parte superior

### 2. Capturar datos semanales
- **Selecciona la semana** (1 a 5)
- **Bajas Reales:** Ingresa las bajas que ya aplicaron esta semana
- **Bajas Mapeadas:** Ingresa las bajas que tienes comprometidas para semanas restantes
- El sistema calcula automáticamente la proyección

### 3. Monitorear estado
- **Verde (✓):** Dentro del objetivo
- **Amarillo (!):** Cuidado, cerca de la meta
- **Rojo (⚠):** Sobre el objetivo, riesgo alto

### 4. Guardar y exportar
- **💾 Guardar Cambios:** Guarda los datos en el sistema
- **📥 Exportar CSV:** Descarga los datos para enviar a RH

## 📊 Métricas que ves

- **Bajas Aplicadas:** Total de bajas reales hasta hoy
- **Objetivo Mes:** Meta de bajas para tu sitio
- **Proyección Mes:** Bajas reales + mapeadas (estimado a cierre)
- **% Avance:** Qué porcentaje de la meta ya alcanzaste

## ❓ Preguntas Frecuentes

**¿Puedo ver otros sitios?**
No, cada líder solo ve y edita su sitio.

**¿Qué pasa si pierdo mis datos?**
Se guardan en la base de datos cada vez que das click en "Guardar Cambios". Si cierras sin guardar, se pierden cambios de esa sesión.

**¿Puedo cambiar contraseña?**
Contacta al coordinador de RH (coordinador.rh@danone.com)

**¿Hay un histórico?**
Sí, cada cambio queda registrado. Puedes exportar CSV para tus registros.
